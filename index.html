HTML

  • HTML stands for Hypertext Markup Language.
  • HTML is a type of markup language used in the creation of webpages and web applications.
  • HTML consists of elements or tags that establish both the structure and content of a webpage.
  • HTML tags are enclosed by angle brackets.
  • HTML attribute provides supplementary information about an element.
  • HTML is structed into different sections, such as the head and body elements, which both play significant roles in the composition of a webpage.
  • The head element contains inforamtion about the webpage.
  • The body element represent the visible content shown to the user.

    Git

    • Below are some common examples of git commands. Exlclude the square brackets (i.e., [ ]). Quotations must be included.
    • git config --global user.name "[your name]": set up a name of the person writing the code
    • git config --global user.email "[your email]": set up an email address associated to the person writing the code
    • git status: checks what branch we are currently on
    • git checkout -b [branch-name]: creates a new branch and switches to it
    • git add -A: adds all modifications in the current branch to the staging area
    • git commit -m "[ descriptive message about the commit]": commit content in the staging area with a message
    • git pull [source] [branch]: sync local branch with the remote branch (i.e., merge any new content from the remote branch to the local environment)
    • git push [source] [branch]: update local commits with the remote branch (i.e., send commits from the local environment to the remote branch)
